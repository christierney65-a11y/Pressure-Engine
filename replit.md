# Pressure Engine

## Overview
A starter project imported from GitHub. Currently set up as a simple static website ready for development.

## Project Structure
- `index.html` - Main landing page
- `server.py` - Python development server (serves static files on port 5000)
- `.gitignore` - Git ignore rules for Python projects

## Development
The project uses a Python HTTP server for development:
- Host: 0.0.0.0
- Port: 5000

Run the server with: `python server.py`

## Deployment
Configured for static deployment serving files from the root directory.
